## ğŸ” Project Overview  
This is a React + Tailwind CSS application built to demonstrate CRUD-free utilities:
- A **Text Translator** that converts English to multiple languages using a free API  
- A **Random String Generator** for creating alphanumeric tokens  
- Clean **client-side routing** between pages: Home, Translator, and Random Generator  

---

## ğŸš€ Features

- **Text Translator**  
  - Translate English text into Hindi, Urdu, Arabic, Spanish, and more  
  - Uses MyMemory API for free translation  
  - Real-time translation with React Hooks (`useState`, `useEffect`, `useCallback`)  

- **Random String Generator**  
  - Generate secure random strings  
  - Customizable length  
  - Copy-to-clipboard support  

- **Client-Side Routing**  
  - React Router DOM (v6)  
  - Clean navigation with a `Navbar` component
